# Archived legacy migrations

Эти миграции выведены из active path в рамках hard reset архитектуры.

Причина:
- старая схема БД больше не является целевой;
- новая схема будет строиться отдельно, с reset-first rollout;
- эти файлы сохраняются только как историческая справка.

Важно:
- не возвращать эти миграции обратно в console/migrations;
- не смешивать их с новой migration chain.
